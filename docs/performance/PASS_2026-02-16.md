# Performance Pass (2026-02-16)

## Scope
- Animation/render loop hotspots in `DiceCubeView`
- Preset menu icon generation in `ViewController`

## Changes
- Added `DiceCubeView` mesh cache keyed by side count and rounded side length to avoid rebuilding SceneKit geometry for repeated layouts.
- Reused cached geometry per update pass instead of rebuilding per die node.
- Added badge image cache keyed by value + badge size to avoid rerendering label textures every update.
- Added per-label value tracking so badge textures update only when the die value changes (or size changes).
- Added preset icon cache in `ViewController` keyed by dice count + mode to avoid repeated UIGraphics work when rebuilding preset menus.
